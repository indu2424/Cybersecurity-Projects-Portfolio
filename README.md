# 🛡️ Cybersecurity Projects Portfolio

## 📌 Overview

This repository is a comprehensive collection of **15 cybersecurity projects** developed as part of my internship and academic learning.
Each project focuses on real-world security concepts such as **network security, cryptography, web security, malware analysis, and threat intelligence**.

The goal of this portfolio is to demonstrate **practical skills in identifying, analyzing, and mitigating cyber threats**.

---

## 🚀 Projects Included

### 🔍 1. Web Vulnerability Scanner

Detects common web vulnerabilities such as SQL Injection and XSS.

### 🔥 2. Smart Personal Firewall

Monitors and controls incoming/outgoing network traffic using custom rules.

### ⌨️ 3. Keylogger with Encrypted Data Exfiltration

Captures keystrokes securely and encrypts logs for safe transmission.

### 🔐 4. Password Strength Analyzer & Wordlist Generator

Analyzes password strength and generates custom attack wordlists.

### 📁 5. Secure File Storage System (AES)

Encrypts and decrypts files using AES encryption.

### 🎣 6. Ethical Phishing Simulation Platform

Simulates phishing attacks for awareness and training purposes.

### 🌐 7. Network Packet Sniffer with Alert System

Captures network packets and detects suspicious activity.

### 🛡️ 8. Linux Hardening Audit Tool

Performs security audits and checks system vulnerabilities.

### 💉 9. SQL Injection Playground

A controlled environment to learn and test SQL injection attacks.

### 💬 10. Secure Chat Application (E2EE)

Implements end-to-end encrypted messaging using RSA & AES.

### 📊 11. Log Analyzer for Intrusion Detection

Analyzes logs to detect brute force and DoS attacks.

### 🚫 12. Browser Tracker Blocker Extension

Blocks tracking scripts and enhances user privacy.

### 🖼️ 13. Steganography Tool

Hides and extracts secret messages inside images.

### 🍯 14. Honeypot Server

Simulates vulnerable services to capture attacker behavior.

### 📡 15. Cyber Threat Intelligence Dashboard

Fetches real-time threat data using APIs and visualizes attack insights.
